About

The vAmiga Test Suite is a specialized collection of Amiga example programs, originally created to improve the accuracy of the vAmiga emulator. Most test cases are provided as bootable ADF images, which can be used both in emulators and on real Amiga machines.

In addition to the ADF images, each test case includes one or more screenshots captured from a real Amiga system.
